# Data Science Learning Project
## Project Title
Student Performance Report Automation

## Description
This project generates a PDF report of student performance using Python.
It reads subject-wise scores from a CSV file and automatically creates:
- A table of scores
- A performance summary
- A monthly PDF report

## Files in This Project
- README.md : Project explanation
- scores.csv : Input data file
- project_code.py : Python code to generate the report
- Sample PDF report (optional)

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- ReportLab
- Google Colab

## How to Use
1. Update scores in `scores.csv`
2. Run the Python code
3. A PDF report is generated automatically
